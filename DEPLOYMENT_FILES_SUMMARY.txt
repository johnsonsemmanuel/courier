DEPLOYMENT WORKFLOW:
================================================================================

1. EXTRACT LOCALLY
   - Unzip courier-savings-bank-deployment.zip on your computer

2. UPLOAD TO CPANEL
   - Upload everything EXCEPT public folder → public_html/banking-app/
   - Upload public folder contents → public_html/

3. CREATE DATABASE
   - Create MySQL database in cPanel
   - Import database_mysql_fixed.sql via phpMyAdmin

4. CONFIGURE
   - Create .env file in public_html/banking-app/
   - Update database credentials
   - Update email settings
   - Update APP_URL

5. FINALIZE
   - Update public_html/index.php paths
   - Create .htaccess files
   - Set folder permissions
   - Test the application

================================================================================
TROUBLESHOOTING QUICK REFERENCE:
================================================================================

500 Internal Server Error:
- Check .env file exists and is configured
- Check storage/ and bootstrap/cache/ permissions (775)
- Check error logs in cPanel

Blank White Page:
- Check index.php paths are correct
- Verify vendor/ folder was uploaded
- Check PHP version (needs 8.1+)

CSS/JS Not Loading:
- Verify public/build/ folder was uploaded
- Check APP_URL in .env matches your domain
- Clear browser cache

Database Connection Error:
- Verify database credentials in .env
- Check database user has privileges
- Confirm database exists

Emails Not Sending:
- Verify email account exists in cPanel
- Check MAIL_* settings in .env
- Test email in cPanel webmail
- Check spam folder
